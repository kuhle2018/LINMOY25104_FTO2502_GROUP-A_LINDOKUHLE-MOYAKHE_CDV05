# 💼 LINDOKUHLE MOYAKHE — Responsive Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript. Designed to showcase web and visual design skills, recent work, testimonials, and contact functionality using EmailJS.

---

## 🚀 Features

- Responsive layout for all devices
- Smooth scroll navigation
- Modal popups for service details
- Swiper.js testimonial carousel
- Scroll-up button visibility on scroll
- Dark/light theme toggle with local storage
- Contact form integration via EmailJS
